Soft Mod Examples

Original Xbox

  • Requirements: Splinter Cell, USB Stick
  • Exploit: Savegame exploits buffer underflow in font loader

Playstation 3

  • Requirements: Internet Access, USB Stick
  • Exploit: Webkit exploit, which allows code run for Kernel exploit
  • Notes: Works on "Phat" and Slim PS3s. Similar hack in-the-works for "Super-slims"

Nintendo 3DS

Method 1

  • Requirements: SD Card, Sound File, Firmware 9.0.0 - 11.3.0
  • Exploit: Heap overflow in sound tag processing found in Sound Player built into OS

Nintendo 3DS

Method 2

  • Requirements: Original DS Flash Card, Magnet
  • Exploit: Intentional backdoor in bootrom for official repairs
  • Notes: Works on any firmware, including current

Playstation 4

  • Requirements: Internet Access, Firmware 4.55 or prior
  • Exploit: Webkit exploit, which allows code run for Kernel exploit
  • Notes: No boot-time exploit, needs to be rerun after every reboot
